# markov-chain
Implementation of a markov chain in python from scratch

The given code implements a markov chain consisting of three states "Hot", "Cold", "Tepid" with the following transition table.

|| Hot | Cold | Tepid |
| --- | --- | --- | --- |
| Hot | 0.2 | 0.6 | 0.2 |
| Cold | 0.1 | 0.6 | 0.3 |
| Tepid | 0.2 | 0.7 | 0.1 |
